# 식사할 상대를 찾아주는 매칭 서비스 < 한끼가치 >

### 먹는 시간은 소중합니다!
##### "혼자 먹긴 심심할 때, 이것저것 다양하게 먹고싶을 때, 당장 친구 부르긴 힘들 때"
1인 가구를 타겟으로 같이 식사할 수 있는 상대를 간편하게 찾을 수 있는 커뮤니티 웹사이트<br>
한끼의 가치에서 비롯되어, 같이 가치있는 순간을 즐기기 위한 서비스 입니다.
